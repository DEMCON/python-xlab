Introduction¶

Xlab is a Python module that facilitates performing lab experiments. Xlab consists of two submodules: the instruments module that allows interfacing to numerous instruments, and the experiments module that facilitates performing experiments by providing measurement, plotting and logging functions.

Installation of prerequisites¶

In order to use the instruments module, the following dependency need to be installed:

pip install pyvisa
