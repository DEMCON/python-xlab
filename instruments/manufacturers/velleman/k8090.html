Velleman K8090 / VM8090 8-channel relay card¶

This page describes the API (programmers interface) for this instrument. Information on how to connect to this instrument is given in the documentation of the instruments module

The Velleman K8090 requires the driver to be installed, which is found here

class xlab.instruments.manufacturers.velleman.k8090.K8090(port)[source]¶
disableButtons()[source]¶

Disable the functionality of the buttons

switchOff(channels)[source]¶

Switch off relays

See switchOn for details

switchOn(channels)[source]¶

Switch on relays

Specify the relays to be switched on either as list of channel numbers (1..8) e.g. [1, 4, 6] or as a bitmask (0..255)

examples:

k8090.switchOn([1,4,6])
k8090.switchOn(0x31)
