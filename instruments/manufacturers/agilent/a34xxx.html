Agilent 34xxx Multimeters¶

This page describes the API (programmers interface) for this instrument. Information on how to connect to this instrument is given in the documentation of the instruments module

class instruments.manufacturers.agilent.a34xxx.A34xxx(interface_id)[source]¶

Common parameters used for most measurement functions:

range: float or str, default ‘AUTO’
The measurement range, a float or ‘MIN’, ‘MAX’, ‘AUTO’, or ‘DEF’
resolution: float or str, default ‘DEF’
The measurement resolution, a float or ‘MIN’, ‘MAX’, or ‘DEF’. A higher resolution results in a higher measurement time
channel: integer, default 1
Some multi-channel multimeters (e.g. the 34450) support a secondary display value. Specify channel=2 to use this secondary display

For range and resolution, the float values ‘inf’ and ‘-inf’ are interpreted as ‘MAX’ and ‘MIN’, respectively
