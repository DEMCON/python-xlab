Agilent 335xx Function generator¶

This page describes the API (programmers interface) for this instrument. Information on how to connect to this instrument is given in the documentation of the instruments module

class instruments.manufacturers.agilent.a335xx.A335xx(interface_id)[source]¶
apply(function, frequency, amplitude, offset=0.0)¶

Set the function generator function, frequency [Hz], amplitude [Vpp] and offset cancel any modulation, enable output

function: any of: ‘SIN’, ‘SQUARE’, ‘RAMP’, ‘PULSE’, ‘NOISE’, ‘DC’, ‘USER’

load_waveform(wave, activate=True)¶

Load a waveform specified by wave. Wave is an iterable of floats between -1.0 and 1.0

If activate == True (default), select the waveform and select the ‘user’ function

select_user_waveform(name)¶

Name should be any of EXP_RISE, EXP_FALL, NEG_RAMP, SINC, CARDIAC, VOLATILE

set_amplitude_vpp(voltage)¶

Set the output amplitude [V peak-peak]

set_amplitude_vrms(voltage)¶

Set the output amplitude [V rms]

set_frequency(frequency)¶

Set the output frequency [Hz]

set_function(function)¶

Set the function generator function

function: any of ‘SIN’, ‘SQUARE’, ‘RAMP’, ‘PULSE’, ‘NOISE’, ‘DC’, ‘USER’

set_load(load)¶

Set the output load. Can be a float [Ohm] or ‘INF’, ‘MIN’, ‘MAX’. Note: for consistency, a float value of ‘+inf’ corresponds to ‘MAX’, and thus not to ‘INF’

set_offset(voltage)¶

Set the output offset [V]

set_period(period)¶

Set the period [s], this is the inverse of the frequency

set_pulse_dutycyle(dutycycle)¶

Set the duty cycle for pulse function [0..1]

set_pulse_transition(transition)¶

Set the transition time for the pulse function [s]

set_pulse_width(width)¶

Set the pulse width for the pulse function [s]

set_ramp_symmetry(symmetry)¶

Set the symmetry for ramp function [0..1]

set_square_dutycycle(dutycycle)¶

Set the duty cycle for square wave function [0..1]

set_text(text=None)¶

Set the given text on the display. text==None (default) restores normal display

set_voltage_high(voltage)¶

Set the output high level voltage [V]

set_voltage_low(voltage)¶

Set the output low level voltage [V]
